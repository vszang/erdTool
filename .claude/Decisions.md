## 2026-04-19

### [ADR-006] 하트비트(Heartbeat) 기반 수명 주기 관리
**결정**: 브라우저 프론트엔드에서 5초마다 백엔드로 핑을 보내고, 백엔드는 15초 이상 신호가 없으면 스스로 종료  
**이유**:
- Standalone EXE는 사용자가 종료 버튼을 누르기 어려움 (콘솔 숨김 모드)
- 브라우저 탭을 닫았을 때 백그라운드 프로세스가 남는 고질적 문제 해결
- 시스템 자원 낭비 방지
**포기한 대안**:
- 수동 종료 버튼: 사용자가 잊을 수 있음
- OS 프로세스 감시: 구현 복잡도 및 권한 문제

### [ADR-007] CSS 변수 기반 테마 커스터마이징
**결정**: 모든 시각적 요소(배경, 텍스트 색상 등)를 CSS 변수로 관리하고 Vue의 reactive store와 연동  
**이유**:
- 사용자가 실시간으로 디자인 변경 사항을 확인 가능해야 함
- `localStorage`를 통한 설정 유지 용이
- 하드코딩된 스타일을 배제하여 유지보수성 향상
**포기한 대안**:
- 인라인 스타일: 코드 중복 및 성능 이슈 가능성
- 전역 클래스 교체: 조합의 수가 너무 많음 (색상별 클래스 생성 불가)

### [ADR-008] 커맨드라인 인자 기반 포트 지정
**결정**: `argparse`를 사용하여 `--port`(-p) 인자를 처리하고, 브라우저 오픈 로직을 이에 연동  
**이유**:
- 5000 포트가 선점된 환경에서 충돌 가능성 대비
- 사용자에게 유연한 실행 환경 제공
**포기한 대안**:
- 환경 변수(ENV): 로컬 실행 시 사용자 편의성 낮음
- 설정 파일: 단순 포트 변경을 위해 파일을 수정하는 것은 번거로움
